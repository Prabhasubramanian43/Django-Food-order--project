# Food Ordering System

This is a simple online food ordering web application built using Django.

## Features
- View food menu
- Add food items
- Place orders
- Manage orders

## Technologies Used
- Python
- Django
- HTML
- CSS

## Installation

1. Clone the repository
2. Open the project folder
3. Run the development server

Command:

python manage.py runserver

Then open in browser:

http://127.0.0.1:8000/
