# Django Food Order Project

A simple Food Ordering Website using Django and SQLite.

## Features
- Home Page
- About Page
- Menu Page
- Order Page
- SQLite Database
- Image Upload
- Django Admin

## Technologies
- Python
- Django
- SQLite
- HTML
- CSS

## Run Project

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
